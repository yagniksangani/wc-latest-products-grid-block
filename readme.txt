WooCommerce Latest Products Grid Block.

== Description ==

Display the latest woocommerce products with this custom gutenberg block.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/wc-latest-products-grid-block` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
